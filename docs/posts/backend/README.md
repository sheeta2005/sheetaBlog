# 后端专栏

欢迎来到后端技术学习专栏，这里记录了我在后端开发过程中的学习笔记和实践经验。

## 📚 技术栈

### Java 核心
- Java 基础与集合框架
- 并发编程（JUC）
- JVM 原理与调优

### 框架与中间件
- Spring Boot / Spring Cloud
- MyBatis-Plus
- 消息队列（RabbitMQ、Kafka）

### 数据库
- MySQL 优化与索引
- Redis 缓存策略
- 分布式锁与事务

### 工程化
- Docker 容器化
- CI/CD 自动化部署
- 系统设计与架构

## 🚀 学习建议

后端开发注重**系统性思维**，建议按以下顺序学习：

1. Java 基础 → 2. 数据库 → 3. 框架 → 4. 中间件 → 5. 分布式

> 所有内容均基于实际项目经验总结，持续更新中...
