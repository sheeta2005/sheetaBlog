# 前端专栏

欢迎来到前端技术学习专栏，这里记录了我在学习前端开发过程中的笔记和心得。

## 📚 技术栈

### 核心框架
- Vue3 组合式 API
- TypeScript 类型系统
- Vite 构建工具

### 生态工具
- Pinia 状态管理
- Vue Router 路由
- Element Plus UI 组件库

### 工程化
- ESLint + Prettier 代码规范
- Vitest 单元测试
- 性能优化与实践

## 💡 为什么后端要学前端？

全栈思维能帮助我们：
- 更好地理解业务需求
- 提升前后端协作效率
- 独立完成完整项目

> 从后端视角出发，聚焦实用技能，快速上手前端开发。
