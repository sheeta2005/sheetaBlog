# 算法专栏

这里记录了常见算法的实现、讲解以及 LeetCode 刷题心得。

## 📚 算法分类

- **搜索算法**：BFS、DFS
- **字符串匹配**：KMP、Rabin-Karp
- **高级数据结构**：线段树、树状数组
- **动态规划**：背包问题、区间 DP
- **图论算法**：最短路径、最小生成树

1. 基础算法 → 2. 数据结构 → 3. 高级算法 → 4. LeetCode 实战
